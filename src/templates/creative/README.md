# Creative 创意风格模板

## 设计理念

现代创意风格，适合互联网、设计、创意类岗位，展现个性和创新能力。

## 视觉特点

### 布局
- **卡片式设计** - 每个block都是独立圆角卡片
- **不对称布局** - 灵活的视觉节奏
- **渐变背景** - 灰色渐变底色，提升层次感

### 样式
- **圆角阴影** - 大圆角（rounded-2xl）+ 阴影效果
- **彩色点缀** - 主题色圆点装饰
- **浅色底色** - 卡片内使用灰色50底色

### 特色元素
- ✅ 渐变背景（from-gray-50 to-gray-100）
- ✅ 圆角头像卡片（带渐变装饰）
- ✅ 求职意向3栏卡片布局
- ✅ Section标题竖线装饰
- ✅ Block左侧主题色边框
- ✅ 时间胶囊样式（圆角背景）
- ✅ 悬浮阴影效果

## 适用场景

### ✅ 适合
- 互联网公司
- 设计师岗位
- 创意类岗位
- 产品经理
- UI/UX设计
- 前端开发
- 创业公司
- 年轻化团队

### ❌ 不太适合
- 传统保守行业
- 高管岗位

## 技术实现

### 使用新架构
完全采用可复用的 Wrapper 组件：

```tsx
// 卡片内容
<div className="bg-white rounded-2xl shadow-md p-5 hover:shadow-lg transition-shadow border-l-4">
  {/* 字段编辑 */}
  <EditableFieldWrapper
    blockId={block.id}
    fieldName="company"
    value={block.company}
    onUpdate={() => {}}
  />
  
  {/* 富文本编辑 */}
  <EditableBlockWrapper
    blockId={block.id}
    contentField="contentHtml"
    contentSize="xs"
  />
</div>
```

### 响应式设计
- ✅ 卡片悬浮效果（hover:shadow-lg）
- ✅ 平滑过渡动画（transition-shadow）
- ✅ 主题色动态应用

## 设计细节

### 颜色系统
```tsx
// 主题色用法
background: `linear-gradient(135deg, ${themeColor}, ${themeColor}99)`  // 头像渐变
backgroundColor: `${themeColor}15`  // 时间标签背景（15% 透明度）
borderColor: theme.primaryColor      // 卡片左边框
```

### 圆角系统
```tsx
rounded-2xl  // 大圆角（卡片）
rounded-full // 全圆（装饰点、时间标签）
rounded-lg   // 中圆角（内容区域）
```

### 阴影系统
```tsx
shadow-xl    // 整体容器
shadow-lg    // 头像、悬浮卡片
shadow-md    // 普通卡片
```

## 自定义建议

### 调整卡片圆角
```tsx
// 更圆润
className="rounded-3xl"

// 更方正
className="rounded-xl"
```

### 调整间距
```tsx
// 更紧凑
className="space-y-3"

// 更宽松
className="space-y-8"
```

### 调整背景
```tsx
// 纯白背景
className="bg-white"

// 更浅的渐变
className="bg-gradient-to-br from-gray-25 to-white"
```

## 配色建议

### 主题色推荐
- **蓝色** `#3B82F6` - 专业、科技感
- **紫色** `#8B5CF6` - 创意、艺术感  
- **绿色** `#10B981` - 活力、成长
- **橙色** `#F59E0B` - 热情、创新

### 搭配原则
- 主题色用于：标题、装饰、边框
- 灰色用于：背景、辅助文字
- 白色用于：卡片背景、主要内容
