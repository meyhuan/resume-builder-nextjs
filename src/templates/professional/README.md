# Professional 专业商务模板

## 设计理念

传统商务风格，适合正式求职场合，特别适合金融、法律、管理等传统行业。

## 视觉特点

### 布局
- **单栏布局** - 清晰的垂直阅读动线
- **中心对齐标题** - 姓名居中，彰显专业感
- **左边框点缀** - 每个block都有灰色左边框，增强层次感

### 样式
- **正式配色** - 以主题色和黑白灰为主
- **横线分隔** - 使用主题色横线分隔各个section
- **传统排版** - 标题、职位、时间清晰分明

### 特色元素
- ✅ 顶部主题色装饰条
- ✅ 中心对齐的姓名和联系方式
- ✅ 清晰的section标题（主题色 + 横线）
- ✅ 每个block左侧灰色边框
- ✅ 时间放在右侧，突出重点
- ✅ 白色背景，正式商务

## 适用场景

### ✅ 适合
- 传统行业（金融、法律、咨询）
- 管理岗位
- 资深职位
- 正式商务场合
- 保守型企业文化

### ❌ 不太适合
- 创意行业（设计、广告）
- 互联网创业公司
- 艺术类岗位

## 技术实现

### 使用新架构
本模板完全使用新的 EditableBlockWrapper 和 EditableFieldWrapper 组件：

```tsx
// 字段编辑
<EditableFieldWrapper
  blockId={block.id}
  fieldName="company"
  value={block.company}
  onUpdate={() => {}}
  className="font-semibold"
/>

// 内容编辑
<EditableBlockWrapper
  blockId={block.id}
  contentField="contentHtml"
  contentSize="xs"
/>
```

### 零重复代码
- ✅ 无状态管理代码
- ✅ 无编辑逻辑代码
- ✅ 只关注布局和样式
- ✅ 代码量极少（~400行）

## 自定义建议

如果需要调整样式，只需修改 Tailwind 类名：

```tsx
// 调整卡片圆角
className="rounded-lg"  // 改为 rounded-2xl

// 调整边框粗细
className="border-l-2"  // 改为 border-l-4

// 调整间距
className="space-y-4"   // 改为 space-y-6
```
