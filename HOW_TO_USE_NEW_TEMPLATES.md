# 如何使用新模板

## ✅ 新模板已集成完成！

已成功将 **Professional** 和 **Creative** 两个新模板集成到应用中。

## 使用方法

### 1. 启动应用

```bash
pnpm dev
```

### 2. 切换模板

在右侧边栏中：
1. 点击 **"切换模板"** 标签
2. 选择你想要的模板：
   - **简约** - 原有的 Simple 模板
   - **现代** - Modern 模板
   - **专业商务** - 新的 Professional 模板（正式商务风格）
   - **创意风格** - 新的 Creative 模板（卡片式现代风格）

### 3. 编辑内容

所有模板都支持完整的编辑功能：
- ✅ 点击任何字段进入编辑
- ✅ 点击内容区域进行富文本编辑
- ✅ 使用工具栏（B、I、U、列表、对齐、缩进）
- ✅ 点击外部自动保存

### 4. 导出

- **Export PDF** - 导出为PDF
- **Export PNG** - 导出为PNG图片

## 模板特色

### Professional 专业商务
- 🎯 传统商务风格
- 📋 单栏清晰布局
- ⚫ 黑白灰配色
- 🏢 适合金融、法律、管理等正式行业

### Creative 创意风格
- 🎨 现代创意设计
- 🃏 卡片式布局
- 🌈 渐变背景
- 💡 适合互联网、设计、创意岗位

## 模板切换

模板选择会自动保存到 localStorage，下次打开应用会记住你的选择。

## 问题排查

### 如果看不到新模板

1. 确保已刷新浏览器
2. 检查浏览器控制台是否有错误
3. 清除浏览器缓存

### 如果编辑功能不工作

1. 确保已安装依赖：`pnpm install`
2. 确保已安装 `@lexical/list` 包：`pnpm add @lexical/list`

## 技术说明

- 所有新模板都使用可复用的编辑组件
- 零重复代码
- 完全类型安全
- 易于扩展

## 下一步

可以继续添加更多模板，只需：
1. 在 `src/templates/` 下创建新目录
2. 使用 `EditableBlockWrapper` 和 `EditableFieldWrapper`
3. 在 `App.tsx` 和 `right-sidebar.tsx` 中注册

就是这么简单！🎉
