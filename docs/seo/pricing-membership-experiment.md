# 智简简历价格与会员策略验证

更新时间：2026-06-21

## 结论

短期继续守住“免费可完成简历”的品牌承诺，不把 PDF 导出、Markdown 导出、基础模板和基础 AI 生成变成强付费墙。

商业化先验证“效率和深度”：

- JD 深度匹配优化
- 多岗位版本管理
- 高级 AI 润色次数
- AI 简历评分报告
- 高级模板包
- 求职包：简历、自我介绍、面试题、投递话术

## 免费能力边界

这些能力保持免费或有明确免费额度：

- 基础 AI 生成简历
- 在线编辑
- 基础模板
- PDF 导出
- Markdown 导出
- 基础岗位模板页和范文页阅读

原因：这些能力构成“用户能不能完成一份可投递简历”的底线，不应在最后导出环节制造强阻断。

## 增值能力建议

| 能力 | 价值 | 建议形态 |
| --- | --- | --- |
| JD 深度匹配优化 | 直接服务投递转化 | 单次包或会员权益 |
| 多岗位版本管理 | 适合同投多岗人群 | 会员权益 |
| 高级 AI 润色 | 提升表达质量 | 次数包或会员权益 |
| AI 简历评分报告 | 给用户明确优化方向 | 单次包 |
| 高级模板包 | 视觉和场景差异化 | 会员权益 |
| 求职包 | 扩展到面试和投递话术 | 高价值单次包 |

## 默认价格实验

- 单次 JD 深度优化：9.9-19.9 元
- 月会员：19.9-29.9 元
- 早鸟终身会员：69-99 元

## 文案原则

- 免费能完成简历。
- 付费买的是更快、更匹配、更专业。
- 不使用“通过率提升 95%”这类未经证明的数据。
- 不在导出前临时拦截用户。

## 上线前验收

- 免费用户能完成一份简历并导出。
- 付费权益页明确写出免费/付费边界。
- JD 匹配工具先验证使用率和转化，再决定是否深度收费。
- 价格实验必须接转化追踪：页面来源、点击、支付、创建简历、导出。
