# SEO 软文资料库

本目录用于整理 `aijianli.cn` 在高权重社区、导航站和内容平台可发布的中文 SEO / 外链文章。

## 文件索引

- **`seo-monitoring.csv`**
  - 用途：SEO 收录与排名监控表
  - 字段：URL、关键词、百度/Google/Bing 收录状态与排名、点击量、转化入口、提交状态、技术检查、百度/Bing 后台核验项
  - 更新频率：新 URL 发布后 24-72 小时复查一次；稳定后每周更新一次

- **`competitors/upcv-competitive-analysis-2026-06-20.md`**
  - 对象：UP 简历 / UpCV
  - 用途：竞品定位、SEO/AEO、内容矩阵、站外分发、Agent/MCP 与产品借鉴分析
  - 结论：优先补 AEO 对比页、AI 新职业范文库、JD 匹配微工具和站外分发矩阵

- **`pricing-membership-experiment.md`**
  - 用途：价格与会员策略验证
  - 结论：保持基础简历制作和导出免费，优先验证 JD 深度优化、多岗位版本和高级 AI 能力付费

- **`offsite-aeo-distribution-plan.md`**
  - 用途：站外 SEO/AEO 分发计划
  - 渠道：小红书、知乎、掘金/CSDN、V2EX、AI 工具导航、简历工具合集站

- **`01-v2ex-独立开发者免费-ai-简历工具.md`**
  - 平台：V2EX
  - 方向：独立开发者 / 产品发布 / 反套路
  - 目标：获取第一波开发者用户反馈与品牌外链

- **`02-juejin-技术实现-ai-简历编辑器.md`**
  - 平台：掘金 / CSDN
  - 方向：技术实现 / 编辑器 / AI 结构化生成
  - 目标：获取技术社区曝光与自然引用

- **`03-zhihu-推荐什么简历工具.md`**
  - 平台：知乎回答 / 知乎专栏
  - 方向：工具推荐 / 求职效率 / 轻软文
  - 目标：覆盖“简历工具推荐”类搜索词

  - 目标：覆盖“简历没回应”“ATS 简历优化”等长尾词

## 使用建议

- **V2EX**
  - 语言更克制
  - 少营销词
  - 强调个人动机、产品取舍、用户痛点

- **掘金 / CSDN**
  - 增加技术栈、实现难点、架构图或编辑器截图
  - 可以补充：Next.js、React、Prisma、所见即所得编辑、AI 文本转结构化简历

- **知乎**
  - 先回答问题
  - 前 3 段不要急着放官网
  - 以“经验总结 + 解决方案”方式自然带出产品

## 截图插入规范

每篇文章都预留了截图标记，统一格式如下：

```md
[截图建议 1：首页 Hero / 编辑器界面 / AI 生成功能]
```

你后续可以把它替换为：

```md
![首页 Hero 截图](./images/hero.png)
```

或平台支持的图片上传后链接。
